# Python-Part6-字典
字典是一种键（key）值（value）对应的数据类型，用花括号包裹，键值对之间用逗号分开。访问元素方式与其他的数据类型类似。

```py
# 字典建议分行写，这样不仅美观而且可读性好
score = {
    "张三": 99,
    "李四": 98,
    "王五": 97
}
print(score["张三"])  # 99
score["张三"] = 100   # 修改键对应的值
score["李六"] = 100   # 如果字典没有这个键，则创建
print(score["张三"])  # 100
print(score)          # {'张三': 100, '李四': 98, '王五': 97, '李六': 100}
```

下列是常用的函数
- `get(x)`：获取键名为x的值。等价于上面的`score[x]`。
- `keys()`：获取所有的键。
- `values()`：获取所有的值。

需要注意的是，字典的键**通常是唯一的**，如果有重复，那么字典会以最后一个键的值为准，替换掉前面的键。

```py
score = {
    "张三": 99,
    "李四": 98,
    "王五": 97,
    "张三": 100
}
print(score)    # {'张三': 100, '李四': 98, '王五': 97}
```

并且，键的类型不能是列表，否则会语法报错。
