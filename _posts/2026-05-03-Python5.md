# Python-Part5-元组

## 元组（Tuple）
元组与列表类似，但是元组不可修改。而且用的是小括号。下标索引从0开始，元素之间用逗号分开。
```py
a = (1, 2, "Mike")
print(a[0]) # 输出1
```
虽然元组不能修改，但是可以像列表一样用`+`组合。

需要注意的是，如果元组中只有一个元素，需要在改元素后面添加逗号`,`。

```py
# 错误写法
a = (1)

# 正确写法
a = (1, )
```
