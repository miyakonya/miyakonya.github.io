# Python-Part2-流程控制语句

## 一.判断
Python以缩进表示代码块。缩进为4个空格（1个tab）。
在Python中，`if`可用于判断，用法为：

```py
if 条件:
    代码，前面必须缩进4个空格，可以用一个tab
```

例如：

```py
a = 10
if a == 10:
    print("a的值是10")
```

输出结果：`a的值是10`
`elif`则是多分支判断，**它必须在if之后，不能单独用**
错误用法：

```py
a = 10
elif a == 10:
    print("a的值是10")
```

正确用法：

```py
a = 10
if a != 10:
    print("a的值不是10")
elif a == 10:
    print("a的值是10")
```

`else`则是所有条件不成立时则执行

```py
a = 5
if a > 5:
    print("a大于5")
elif a < 5:
    print("a小于5")
else:
    print("a=5")
```

输出结果：`a=5`

## 二、循环
Python中有`while`循环和`for`循环。

### while循环
`while`循环语法为：

```py 
while 条件:
    代码，前面必须缩进4个空格
```

打印0-4数字可以用下列代码：

```py
a = 0
while a <= 4:
    print(a)
    a += 1  # 这个运算表示a自增1
```

输出结果：
`
0
1
2
3
4
`

### for循环

`for`循环通常用于遍历一个列表，它的语法为：

```py
for 变量 in 序列:
    代码
```

对于一个列表，可以用下列代码遍历并打印出来：
```py
a = [1, 2, 3]
for i in a:
    print(i)
```
输出结果：`1 2 3`
还能遍历字符串：
```py
string = "Hello"
for char in string:
    print(char)
```

输出结果：
H
e
l
l
o
通常，`for`循环会搭配`range()`函数使用。它可以用于生成整数序列。`range()`函数用法：
`range(x)`：生成0~x的序列
`range(x, y)`：生成x~y的序列
`range(x, y, z)`：生成x~y的序列，步长为z
**注：`range()`函数生成的序列的特点是含头不含尾。**
例如：

```py
for i in range(3):
    print(i)    # 输出1 2 3
for i in range(1, 5):
    print(i)    # 输出1 2 3 4
for i in range(0, 11, 2):
    print(i)    # 输出0 2 4 6 8 10
```

### 嵌套循环
嵌套循环是指循环之间嵌入另一个循环，用于完成复杂的操作
例如：

```py
list1 = [1, 2, 3]
list2 = ['a', 'b', 'c']

for num in list1:
    for char in list2:
        print("组合:", num, char)
```

输出：
组合: 1 a
组合: 1 b
组合: 1 c
组合: 2 a
组合: 2 b
组合: 2 c
组合: 3 a
组合: 3 b
组合: 3 c

### break与continue语句
这两个语句用于控制循环，`break`是**中断当前循环**，`continue`是**直接进入下一次循环**。

```py
for i in range(5):
    if i == 3:
        break
    print(i)
```

输出：`0 1 2`

```py
a = [1, 2, 3]
for i in a:
    if i == 2:
        continue
    print(i)
```

输出：`1 3`
