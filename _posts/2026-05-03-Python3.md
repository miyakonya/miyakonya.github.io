# Python-Part3-字符串

## 一、格式化字符串
目前Python中有三种字符串格式化的方法。
### % 格式化
类似于C语言的`printf()`，占位符需要严格遵循数据类型。

```py
name = "John"
age = 18
print("name: %s, age: %d" % (name, age))
```

输出：`name: John, age: 18`
其中，`%s`表示字符串格式，`%d`表示整数格式。

### format()函数
字符串中提供了`format()`函数用于格式化，这比`%`格式化简单许多。它使用`{}`占位。

```py
name = "John"
age = 18
print("name: {}, age: {}".format(name, age))
```

输出结果同上。

### f-string
在Python 3.6+的版本，提供了一个更加简洁的方法。它可以直接在`{}`中写表达式。

```py
name = "John"
age = 18
print(f"name: {name}, age: {age}")
```
输出结果同上。

## 七、字符串切片
Python可以用方括号截取字符串。
- `[x]`：第n个字符，从0开始。
- `[x:y]`：第x~y个字符，不含x。
- `[x:]`：第x个字符到末尾，不含x。
- `[:x]`：前x个字符

如果x为负数，则从后面开始截取。

```py
message = "Hello Python"
print(message[1])   # 输出e
print(message[1:5]) # 输出ello
print(message[1:])  # 输出ello Python
print(message[:1])  # 输出H
print(message[:-3])  # 输出Hello Pyt
```
