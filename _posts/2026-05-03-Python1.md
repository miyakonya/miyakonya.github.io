# Python-Part1-基础语法
本篇总结一下Python基础知识。包括输入输出，基本数据类型等。
**注：这里只讲Python3**

## 一、输入与输出
在Python中，`print()`函数用于**打印数据到控制台**，而`input()`函数则是**从控制台接收数据**。
`input()`返回的数据类型默认是**str字符串类型**，同时可以接收一个参数作为输入提示符。
`print()`函数**默认结束符是`\n`**，也就是换行符，如果不想换行可以指定end参数，如：

```py
name = input("请输入你的名字:")
print("Hi", name, end='')
```

---

## 二、变量与类型
在Python中，基础数据类型有
- `int`整数型
- `float`浮点型
- `str`字符串型
- `bool`布尔类型
- `list`列表类型
- `tuple`元组类型
- `dict`字典类型

定义一个变量，直接用`"变量名 = 数据"`即可。例如：

```py
num = 10
score = 98.5
name = "John"
flag = True
lst = [1, 2, 3]
host = ("192.168.1.1", 80)
user = {"Mike": 10, "Simon": 1}
```

变量名**必须以字母或下划线开头**，对**大小写敏感**（如count和Count是两个不同的变量），不能用关键字作为变量名。
> 变量名尽量见名知意。
> 全大写的变量名通常表示常量。

## 三、运算符
### 算数运算符
- \+ 加法
- \- 减法
- \* 乘法
- / 除法
- % 模运算，返回余数
- \*\* 幂运算
- // 整除(向下取整)

代码如下：

```py
a = 10
b = 5

print(a + b)    # 输出15
print(a - b)    # 输出5
print(a * b)    # 输出50
print(a / b)    # 输出2.0，浮点数类型
print(a % b)    # 输出0，因为可以整除
print(a ** b)   # 输出100000
print(a // b)   # 输出2
```

### 比较运算符
- == 等于
- != 不等于
- \> 大于
- < 小于
- \>= 大于等于
- <= 小于等于

以上运算符均返回布尔值

```py
a = 10
b = 5

print(a == b)    # 输出False
print(a != b)    # 输出True
print(a > b)    # 输出True
print(a < b)    # 输出False
print(a >= b)    # 输出True
print(a <= b)   # 输出False
```

### 逻辑运算符
- `and`：两边条件均成立。
- `or`：两边条件其中一个成立。
- `not`：取反操作。表达式为`True`时返回`False`，反之才返回`True`。

代码如下：

```py
x = 10
b = 5
flag = True

print(x > 5 and b < 6)  # True
print(x > 5 or b > 10)  # True
print(not flag)         # False
```

## 四、常用内置函数
- `len(x)`：获取变量x的长度。
- `type(x)`：获取变量x的类型。
- `id(x)`：获取变量x的内存地址。
- `max(x)`：获取x中最大的值（一般是列表、元组等数据类型）。
- `min(x)`：获取x中最小的值（一般是列表、元组等数据类型）。
- `del`：删除，严格来说不是函数而是一个语句。

代码如下：

```py
lst = [1, 2, 3]
name = "Mike"

print(id(lst))
print(type(lst))    # <class 'list'>
print(len(lst))     # 3
print(len(name))    # 4
print(max(lst))     # 3
print(min(lst))     # 1
del lst[0]
print(lst)          # [2, 3]
```

## 五、注释
注释是一种写给开发人员看的信息，它不会被程序读取。当代码复杂时，可以写注释注明代码，以方便其他人能看懂。

Python中有单行注释和多行注释，单行注释以`#`开头，多行注释的内容要被三引号包裹，如`'''这是注释'''`。

同时，还可以用注释指定编码，通常写在Python文件顶端：

```py
# -*- coding: UTF-8 -*-
# 上面的表示此文件用UTF-8编码，也可以写成:
# coding: UTF-8

# 单行注释
"""
多行注释
"""

'''
单引号也可以
'''
```

## 六、空语句
由于Python的缩进很严格，所以如果某一个代码块目前还没有代码时，为了不让程序报错，可以用`pass`语句作为占位语句，**它不会做任何事**。

## 七、类型转换
Python中有隐式类型转换和显示类型转换。
隐式类型转换是Python自动帮我们转换的，是为了防止数据丢失。例如，使用一个`int`类型的数加一个`float`类型的数，得到的新值是`float`类型的。

```py
num1 = 3
num2 = 0.2
num3 = num1 + num2
print(type(num1))   # <class 'int'>
print(type(num2))   # <class 'float'>
print(type(num3))   # <class 'float'>
```

而显式类型转换则是要我们自己手动转换。用`数据类型(变量)`即可。

```py
lst = [1, 2, 3]
print(type(lst))    # <class 'list'>
lst2 = str(lst)
print(type(lst2))   # <class 'str'>
```

需要注意的是，**并非所有数据类型都能互相转换**。例如，`str`类型是无法转换为`int`类型的，这会导致报错。而`int`类型可以转换为`str类型`。

```py
a = "Mike"
b = 10
# print(int(a)) 这个代码会报错
print(str(10))
```
