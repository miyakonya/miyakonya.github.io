# Python-Part4-列表

## 列表（List）
列表是Python中的一种常用的数据类型。列表中可以有多种数据类型，也可以重复某一个元素。索引从0开始。可以用方括号访问某个元素，也可以像字符串那样切片访问。

```py
a = ["John", 1, ["1"]]
print(a)    # 输出["John", 1, ["1"]]
print(a[0]) # 输出John
print(a[0:2]) # 输出['John', 1]
```

输出：`['John', 1, ['1']]`

下列是列表常用的函数：
- `append(x)`：在列表末尾添加一个元素。
- `remove(x)`：删除x元素。
- `clear()`：清空列表
- `reverse()`：反向排序列表中的元素。
- `insert(x, y)`：将一个y插入到列表的第x个索引位置。
- `count(x)`：统计元素x出现的次数。
- `sort()`：排序。

```py
lst = [1, 2, 3, 2, 2, 3, 3, 5, 10]
lst.append(4)
print(lst)  # 输出[1, 2, 3, 2, 2, 3, 3, 5, 10, 4]
lst.sort()
print(lst)  # 输出 [1, 2, 2, 2, 3, 3, 3, 4, 5, 10]
lst.remove(2)
print(lst)  # 输出[1, 3, 2, 2, 3, 3, 5, 10, 4]
lst.reverse()
print(lst)  # 输出[4, 10, 5, 3, 3, 2, 2, 3, 1]
lst.insert(1, "hi")
print(lst)  # 输出[4, 'hi', 10, 5, 3, 3, 2, 2, 3, 1]
print(lst.count(2)) # 输出2
lst.clear()
print(lst)  # 输出[]
```

两个列表还能用`+`组合：

```py
lst1 = [1, 2]
lst2 = [3, 4]
lst3 = lst1 + lst2
print(lst3)
```

输出结果：`[1, 2, 3, 4]`
